# جمهورك (Jomhorak.com) - المنصة الأولى لخدمات السوشيال ميديا

هذا المشروع هو منصة SMM Panel متكاملة تعمل بنظام NextJS و Firebase.

## كيفية تفعيل الوظائف الحقيقية:

لضمان عمل الموقع بشكل سليم (شحن الرصيد، الطلبات، التسجيل)، يجب تنفيذ الخطوات التالية في لوحة تحكم Firebase:

### 1. قواعد أمان Firestore:
انسخ المحتوى الموجود في ملف `firestore.rules` والصقه في:
`Firebase Console > Firestore Database > Rules`

### 2. تفعيل تسجيل الدخول:
تأكد من تفعيل "Email/Password" في:
`Firebase Console > Authentication > Sign-in method`

### 3. الصلاحيات الإدارية (كيف تصبح Admin):
لتحويل حسابك إلى "مشرف" (Admin) لكي تتمكن من شحن أرصدة المستخدمين وقبول الطلبات:
1. اذهب إلى `Firebase Console > Firestore Database > users`.
2. ابحث عن الوثيقة الخاصة بحسابك (يمكنك الاستدلال بالبريد الإلكتروني).
3. قم بتغيير قيمة الحقل `isAdmin` من `false` إلى `true`.
4. قم بتحديث صفحة الموقع، وستظهر لك "لوحة تحكم المشرف" في القائمة الجانبية.
