# **App Name**: TrendPulse

## Core Features:

- Conversion Landing Page: An introductory interface featuring an agency-style overview of services for platforms like Instagram and TikTok, with client social proof sections.
- Package Intelligence Tool: An AI-powered tool that analyzes the user's social media goals to suggest the most effective service packages.
- Platform-Specific Directory: A sophisticated UI layout for browsing services categorized by platform type with high-quality icons and clear pricing tiers.
- Dynamic Balance Dashboard: A private member area displaying real-time wallet balances, active service statuses, and intuitive performance cards.
- Streamlined Order Wizard: A minimalist multi-step UI form for placing new orders quickly, focused on URL validation and volume input.
- Service Tracking & Ledger: A backend-powered searchable table for viewing comprehensive order histories and detailed logs of all previous activities.
- Affiliate Revenue Portal: A specialized UI view for users to monitor referral bonuses and manage earnings from the site's growth system.

## Style Guidelines:

- The design follows a premium dark scheme centered on high-trust Indigo and Electric Blue.
- Primary Color: Royal Indigo (#8977F2), designed for a vibrant and professional tech aesthetic.
- Background Color: Deep Graphite (#0D0C12), created from a desaturated hue of the primary color to ensure depth and contrast.
- Accent Color: Vibrant Cerulean (#1787FF), providing an analogous but distinct contrast for CTA buttons and highlighted badges.
- Headline Font: 'Space Grotesk', bringing a machined and tech-forward feel to the brand. Body Font: 'Inter' (sans-serif) for high readability in service lists and transaction ledgers.
- Custom outline icons with subtle duotone fills, utilizing different colors to categorize platforms (e.g., magenta for Instagram, cyan for TikTok).
- A contemporary 'glass-card' design layout featuring softly rounded corners (16px radius) and sophisticated shadow layers to distinguish UI elements from the background.
- Subtle entrance stagger animations for dashboard cards and a micro-interaction glowing effect on 'Add Funds' buttons.
